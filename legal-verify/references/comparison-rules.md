## 二、数字与期限比对规则

### 2.1 日期比对

| 比对类型 | 规则 | 示例 |
|---------|------|------|
| 日数差异 | 逐字比对，1日之差即为实质差异 | "30日" vs "60日" → 🔴 严重 |
| 工作日 vs 自然日 | 必须区分 | "15个工作日" vs "15日" → 🔴 严重 |
| 年限差异 | 逐字比对 | "5年" vs "10年" → 🔴 严重 |
| 日期格式 | 确认格式是否一致 | "2024年1月1日" vs "2024-01-01" → 🟢 轻微（格式差异） |

### 2.2 比例与分数

| 比对类型 | 规则 | 示例 |
|---------|------|------|
| 分数差异 | 逐字比对 | "三分之一" vs "三分之二" → 🔴 严重 |
| 百分比差异 | 数值比对 | "10%" vs "20%" → 🔴 严重 |
| 分数 vs 百分比 | 需换算后比对 | "二分之一" vs "50%" → 🟢 等价 |

### 2.3 金额与人数

| 比对类型 | 规则 | 示例 |
|---------|------|------|
| 金额差异 | 逐字比对，特别注意"万""亿" | "5万元" vs "50万元" → 🔴 严重 |
| 人数阈值 | 逐字比对 | "200人" vs "300人" → 🔴 严重 |
| 倍数差异 | 逐字比对 | "2倍" vs "5倍" → 🔴 严重 |

### 2.4 数字表述转换规则

| 中文大写 | 中文小写 | 阿拉伯数字 |
|---------|---------|-----------|
| 壹 | 一 | 1 |
| 贰 | 二/两 | 2 |
| 叁 | 三 | 3 |
| 伍 | 五 | 5 |
| 拾 | 十 | 10 |
| 佰 | 百 | 100 |
| 仟 | 千 | 1,000 |

## 三、法条结构比对规则

### 3.1 条款层级

中国法律采用"条→款→项"三级结构：

```
第X条 → 条（最高层级）
  第X款 → 款（条下的分段）
    （X）→ 项（款下的列举）
```

**比对规则：**
- 用户引用"第X条"但实际内容位于"第X条第Y款" → 🟡 实质差异（引用不精确）
- 用户引用"第X条第1款"但官方只有"第X条"（未分款）→ 🟡 实质差异（过度细分）
- 用户引用的款项编号在官方文本中不存在 → 🔴 严重差异

### 3.2 但书/除外条款

**比对规则：**
- 但书（"但是……"）内容遗漏 → 🔴 严重（改变了法条适用范围）
- 但书内容添加（官方无但书）→ 🔴 严重（缩小了法条适用范围）
- "除外"条款遗漏 → 🔴 严重
- "另有规定除外"遗漏 → 🟡 实质

### 3.3 引用条款

**比对规则：**
- 法条内部引用其他条款（"依照本法第X条的规定"）→ 必须验证被引用条款是否存在且对应
- 修订后条款编号变动 → 被引用条款号可能已失效，需标注

## 四、比对结果分级标准

### 4.1 分级定义

| 级别 | 代号 | 颜色 | 含义 | 对法条适用的影响 |
|------|------|------|------|---------------|
| 完全一致 | ID | 🟢 | 用户文本与官方原文逐字一致 | 无影响 |
| 轻微差异 | MN | 🟡 | 标点、空格、数字格式差异，意思不变 | 无实质影响 |
| 实质差异 | SD | 🟠 | 关键词、数字、主体等差异，可能改变法条含义 | 可能影响适用 |
| 严重差异 | CD | 🔴 | 逻辑连词、模态词、否定词差异，实质改变法条含义 | 不可按此引用 |
| 无法核实 | UV | ⚪ | 未找到官方原文进行比对 | 不确定 |

### 4.2 分级判定流程

```
用户文本 vs 官方原文
│
├─ 逐字完全一致 → 🟢 ID
│
├─ 仅以下差异 → 🟡 MN
│  ├─ 标点符号差异（逗号/句号/分号互换）
│  ├─ 数字格式差异（中文/阿拉伯/大写）
│  ├─ 空格/换行差异
│  └─ 同义表述（"得"="可以"、"须"="应当"）
│
├─ 以下差异 → 🟠 SD
│  ├─ 主体名称差异（"公司"vs"企业"）
│  ├─ 日期/时限差异但幅度小
│  ├─ 款项引用不精确（仅引用"条"未指明"款"）
│  ├─ 非核心词增删
│  └─ 除外条款/但书遗漏或添加
│
└─ 以下差异 → 🔴 CD
   ├─ 逻辑连词互换（"且"↔"或"）
   ├─ 模态词互换（"应当"↔"可以"）
   ├─ 否定词互换（"不得"↔"不应"）
   ├─ 关键数字差异（金额、比例、日数）
   ├─ 主体严重错配（"法人"↔"法定代表人"）
   └─ 法条意思完全相反
```

### 4.3 比对输出格式

对每处差异，按以下格式记录：

```json
{
  "location": "第X条第Y款",
  "user_text": "用户原文片段",
  "official_text": "官方原文片段",
  "diff_type": "ID|MN|SD|CD",
  "diff_category": "关键词|数字|逻辑词|主体|结构|标点",
  "impact": "差异的具体影响说明"
}
```

## 五、特殊比对场景

### 5.1 条文整合（民法典）

原多部单行法条文整合入民法典后，可能出现：
- 内容合并（多条规定合并为一条）
- 内容拆分（一条规定拆分为多条）
- 内容调整（措辞调整但实质未变）

**比对规则**：必须同时标注原法条位置和民法典对应位置。

### 5.2 修正案插入条款

刑法修正案等以"第X条之一""第X条之二"形式插入新条款。

**比对规则**：确认"之一""之二"条款的准确编号和位置，注意"之一""之二"的顺序不可颠倒。

### 5.3 授权性条款

"由国务院规定""由省一级人民政府确定"等授权性条款。

**比对规则**：需确认授权对象是否正确，授权范围是否准确。

### 5.4 法律适用条款

"本法自X年X月X日起施行""本法第X条自X年X月X日起施行"等。

**比对规则**：施行日期必须精确到日，部分条款的施行日期可能与整部法律不同。

## 六、解释性差异（Interpretive Conflict）

### 6.1 问题定义

法条文本与官方原文**完全一致**、法律也**现行有效**，但司法解释或配套规定已改变/明确了该条款的理解方式，导致引用者的法律适用结论错误。

> ⚠️ **v1.2.0 核心认知**：解释性差异的判定**依赖于用户的适用意图**。不知道用户怎么理解该条款，就无法判断司法解释是否与用户理解冲突。

### 6.2 典型案例

**《劳动合同法》第14条第2款第3项**

| 维度 | 内容 |
|------|------|
| 法条原文 | "连续订立二次固定期限劳动合同，且劳动者没有本法第三十九条和第四十条第一项、第二项规定的情形，续订劳动合同的" |
| 律师理解 | "订立"指重新签订劳动合同，"延长协议"仅属变更，不计入"连续订立"次数 |
| 司法解释 | 法释〔2025〕12号第10条：延长劳动合同期限累计1年以上，视为连续订立二次固定期限劳动合同 |
| 关键词级搜索命中 | "劳动合同法 订立 延长 司法解释" ✅ 精准命中 |
| 法条级搜索命中 | "劳动合同法 司法解释 第14条" ⚠️ 可能仅返回笼统结果 |
| 结论 | 法条文本正确、法律现行有效，但律师的法律适用因未注意到司法解释而错误 |

**v1.1.0为什么没发现此问题？**
1. 仅执行法条级搜索，搜索"第14条司法解释"返回的是整部司法解释，无法精准定位
2. 不知道用户的适用意图，无法判断"订立"是否就是分歧点
3. 步骤4.3是步骤4的子步骤，容易被AI跳过

### 6.3 解释性差异分级

| 级别 | 代号 | 含义 | 典型场景 | 判定条件 |
|------|------|------|---------|---------|
| 严重解释性差异 | IC-CD | 司法解释明确否定了引用者的理解 | 司法解释将"订立"明确解释为包含"延长" | 用户提供了适用意图 + 司法解释否定 |
| 实质解释性差异 | IC-SD | 司法解释改变了法条的适用条件 | 司法解释新增了适用例外情形 | 用户提供了适用意图 + 司法解释改变条件 |
| 轻微解释性差异 | IC-MN | 司法解释补充了法条未明确的事项 | 司法解释明确了计算标准 | 司法解释补充而非否定/改变 |
| 提示性信息 | IC-Info | 配套规定细化了适用标准（未改变理解方向） | 实施条例细化了申请时限 | 配套规定细化而非否定/改变 |
| 发现但无法判定 | IC-Pending | 发现相关司法解释但用户未提供适用意图 | — | 用户未提供适用意图，仅报告发现 |

### 6.4 解释性差异与文本差异的关系

**解释性差异（IC）是与文本差异（ID/MN/SD/CD/UV）并行的独立维度。**

一个核验结果可能同时具有：
- 文本差异级别：ID（文本完全一致）
- 解释性差异级别：IC-CD（司法解释否定了用户的理解）

**核心原则：文本完全一致 ≠ 法律适用正确**

### 6.5 判定流程（v1.2.0 双轨搜索版）

```
法条文本比对完成
│
├─ 用户提供了适用意图？
│  ├─ 是 → 提取分歧点关键词 → 执行双轨搜索
│  └─ 否 → 主动询问 → 若仍未提供 → 执行双轨搜索但标记IC-Pending
│
├─ 轨道一搜索（法条级）→ 找到司法解释
│  ├─ 与用户理解一致 → 无解释性差异
│  ├─ 司法解释否定了用户理解 → IC-CD 🔴
│  ├─ 司法解释改变了适用条件 → IC-SD 🟠
│  ├─ 司法解释补充了未明确事项 → IC-MN 🟡
│  ├─ 司法解释细化了适用标准 → IC-Info 🟢
│  └─ 用户未提供意图 → IC-Pending ⚪（仅报告发现，不判定级别）
│
├─ 轨道二搜索（关键词级）→ 找到司法解释
│  └─ 按上述标准判定（关键词级可能命中法条级未命中的结果）
│
├─ 轨道一+二均未找到 → 注明"已执行法条级+关键词级搜索，未发现相关司法解释"
│
└─ 轨道三/四搜索 → 同上判定
```
