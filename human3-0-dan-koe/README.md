# HUMAN 3.0 发展评估技能

## 简介

本技能基于 **Dan Koe** 的 HUMAN 3.0 模型，是一个全面的发展评估工具。它通过自适应访谈，评估个体在四个生命象限中的发展现状，识别其元类型（Metatype）和生活方式原型（Lifestyle Archetype），并以解决问题为导向，提供切实可行的转型策略。

## 模型核心

HUMAN 3.0 模型融合了螺旋动力学、肯·威尔伯的 AQAL 模型、自我发展的九个阶段、心流心理学、古代哲学和现代职业理论，将身、心、灵和天职整合为一个可导航的现代生活蓝图。

### 四大象限

| 象限 | 领域 | 关注点 |
|------|------|--------|
| 🧠 心灵 (Mind) | 个人精神世界 | 思想、信念、元认知、模式识别 |
| 🏛️ 身体 (Body) | 个人物理世界 | 健康、健身、能量、身体习惯 |
| ✨ 精神 (Spirit) | 集体精神世界 | 关系、意义、归属、超越性连接 |
| 💼 天职 (Vocation) | 集体物质世界 | 职业、价值创造、影响力、遗产 |

### 三个意识层级

- **1.0 循规蹈矩者** — 遵循权威与传统，非黑即白思维
- **2.0 个人主义者** — 追求个人目标，反叛但可能误将逆向思维当智慧
- **3.0 合成器** — 整合多元视角，在悖论中发现真理，创造新游戏

## 使用方法

### 触发方式

当你在对话中表达以下需求时，该技能会自动激活：

- 想要评估自己的个人发展状况
- 寻求生活整合/生活方式设计指导
- 想了解自己的成长模式和瓶颈
- 对 HUMAN 3.0 模型感兴趣并希望获得评估

你也可以直接说：

- "帮我做一次 HUMAN 3.0 评估"
- "我想了解我的发展状况"
- "帮我分析我的生活象限"

### 评估流程

1. **开场** — 技能会以中文欢迎语开始，并从心灵象限提问
2. **自适应访谈** — 逐象限提问（心灵→身体→精神→天职），每次只问一个问题，根据你的回答深度动态调整后续问题
3. **深度探测** — 识别虚假转变、活跃渠道、退步模式等特殊情况
4. **跨象限分析** — 发现象限间的阻塞、解锁和隐藏联系
5. **生成报告** — 输出完整的中文评估报告，包含元类型、象限分析、转型策略和漏洞评估

### 评估报告包含

- 你的元类型和生活方式原型
- 四象限详细分析（阶段、水平、优势、差距）
- 跨象限动力学分析
- 核心问题识别
- 30天/90天/6-12个月转型策略
- 漏洞（Glitch）风险评估
- 关键警告和即时行动建议

## 注意事项

- 所有输出均为**中文**
- 评估过程需要逐题回答，请耐心配合
- 评估结果不显示数值分数，而是以描述性方式呈现
- 对于"漏洞"（Glitches，如AI、迷幻剂等加速器），技能会根据你的发展水平给出谨慎的建议
- 模型专有术语（如 Metatype、Glitch、Channel 等）保留英文原文

## 来源

本技能基于 Dan Koe 的 HUMAN 3.0 模型原始英文提示词制作，内容忠实于原版框架。
